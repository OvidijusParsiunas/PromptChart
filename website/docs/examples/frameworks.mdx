## Svelte

The Svelte browser app should work right out of the box. The SvelteKit implementation requires the component to be imported using `onMount` and
`prompt-chart` to be rendered when it has loaded.

## Angular

Please add the `CUSTOM_ELEMENTS_SCHEMA` into your module's `schemas` array and import the component
as `import 'prompt-chart';`. See this [blog](https://www.thisdot.co/blog/how-to-integrate-web-components-using-lit-in-angular) for more information.

[Click for Live Example](https://stackblitz.com/edit/stackblitz-starters-csgenhfe?file=src%2Fapp%2Fapp.component.ts)

## Next

Use the `prompt-chart-react` package and lazy load it using Next's `dynamic` extension. <br />
When using [App Router](https://nextjs.org/docs/app) - make sure to set `"use client"` at the top of your file.

[Click for Live Example - App Router](https://stackblitz.com/edit/deep-chat-nextjs-app-qozzblws?file=app%2Fpage.tsx) <br />
[Click for Live Example - Pages Router](https://stackblitz.com/edit/deep-chat-nextjs-hmfzo5we?file=pages%2Findex.tsx) <br />
[Click for GitHub](https://github.com/OvidijusParsiunas/PromptChart/tree/main/examples/nextjs)

## Troubleshooting

If your platform has not been listed and Vanilla JS approach does not work as fallback, or if you are having issues with any of
the existing examples; please see [**github issues**](https://github.com/OvidijusParsiunas/PromptChart/issues) or create
a [**new issue ticket**](https://github.com/OvidijusParsiunas/PromptChart/issues/new) and we will look into
it as soon as possible.
