### `requestInterceptor`

- Type: (request: [`RequestDetails`](#requestdetails)) => [`RequestDetails`](#requestdetails) | [`Promise<RequestDetails>`](#requestdetails)

Function to intercept and modify outgoing requests before they are sent.

```js
chartElementRef.requestInterceptor = (request) => {
  return {
    ...request,
    headers: {...request.headers, Authorization: 'Bearer token'},
  };
};
```

#### `RequestDetails`

```typescript
interface RequestDetails {
  endpoint: string;
  method: string;
  headers: Record<string, string>;
  body: unknown;
}
```

### `responseInterceptor`

- Type: (response: unknown) => [`ChartResponse`](/docs/data#chartresponse) | [`Promise<ChartResponse>`](/docs/data#chartresponse)

Function to intercept and transform incoming responses before the chart is rendered.

```js
chartElementRef.responseInterceptor = (response) => {
  return response.data || response;
};
```
